# graphql-json-object-type [![Travis][build-badge]][build] [![npm][npm-badge]][npm]
A Json object scalar type for [GraphQL.js](https://github.com/graphql/graphql-js).

## Usage

```js
import JsonObject from 'graphql-type-json';
```
